🪑 How This App Works

Classes

Create one entry per period (New Class). Each class has its own roster, desks, seating, and locks — switching the Class dropdown swaps everything at once. Set your Teacher Name once in the toolbar — it's saved automatically and appears on every printed PDF. Delete Class removes the selected class entirely, including its roster, desks, seating, and locks — this can't be undone.

Adding students

  • Type a name and pick M/F/- (or leave blank), then click +.
  • CSV upload format: Last, First, Gender(M/F) — one student per line. A header row (like "Last Name, First Name, Gender") is detected and skipped automatically.
  • Click the (M)/(F)/(-) badge on any unassigned student's chip to change their gender anytime.
  • Click a student's name (in the Unassigned list or in a seat) to rename them — renaming doesn't move or unseat them.
  • Clear Roster removes every student from the current class at once (and clears seating too) — handy before uploading a whole new roster, since CSV upload only adds students rather than replacing the list.

Building desks

  • Quick Grid: pick rows × columns and seats-per-desk (1 = individual desks, 2 = lab tables), then Generate. This replaces the current layout and clears seating — use it when starting fresh, not for daily reshuffling.
  • + Add Desk: adds one single desk. It's draggable immediately, even before anyone sits there.
  • Once a desk has a student in it, drag it via the Arrange Desks: On toggle instead (this avoids conflicting with dragging the student).
  • Click the × on any desk to remove that table (any seated student returns to Unassigned).
  • Click the rotate icon (↻) to turn a table 90° at a time — the whole table, seats, names, and icons all rotate together, so text stays aligned with whichever way the table is facing. Rotating can shift the table slightly (it turns around its center) — just drag it back into place afterward if needed.
  • Select Tables: On lets you click tables to select them (blue outline). Drag any selected table and the whole selected group moves together, keeping their arrangement. Duplicate Selected clones the whole selected group — same layout, same rotations — placed just to the right, and auto-selects the copies so you can keep duplicating to build a row of matching groups fast. Clear Selection deselects everything.

Saved Layouts (Rows / Groups / etc.)

Build a layout once, then Save Current As... and name it (e.g. "Rows" or "Lab Tables"). Later, pick it from the Saved Layout dropdown and click Apply to drop that same layout into any of your 6 classes. Applying a layout clears current seating for that class. Pick a layout and click Delete to remove it from the list permanently — this only deletes the saved template, not any class currently using that arrangement.

Seating students

  • Drag a student chip from Unassigned onto any open seat.
  • 📌 Place Students: fills every unlocked seat with unlocked students, using whatever's chosen in the Order dropdown, starting from the front row (bottom of the layout) and working back.
  • Order dropdown controls how students get placed: Random (default), or Alphabetical by first/last name — great for a simple, predictable seating chart instead of a shuffle.
  • Alternate gender checkbox: interleaves boy/girl in seat order instead of pure random. Only applies when Order is set to Random.
  • Clear Seats: sends everyone back to Unassigned (does not touch desks).

Seat icons

  • 🔒 / 🔓 (bottom-right of a filled seat): click to lock/unlock that student. Locked students are skipped by Randomize/Reshuffle and protected from drag-and-drop.
  • The undo arrow (top-left of a filled seat): instantly sends that student back to Unassigned — use this if drag-and-drop isn't working well on your device (common on touchscreens/Chromebooks).
IEP / 504 seating: place the student in the seat they need, click 🔒 to lock it, then hit 📌 Place Students as usual — everyone else fills in around them, and they'll stay locked in for future placements too. Locked students are also excluded from the "Alternate gender" calculation, so they won't throw off the balance.

Exporting

📄 Export PDF captures the current layout and automatically picks whichever page orientation prints it biggest — landscape as-is, or rotated 90° onto portrait — whichever fills more of the sheet for your specific layout. It adds the class name, your teacher name (if set), and the date/time as the header. The file name includes the date and time too, so it's easy to tell which export is the newest. Locked seats show a gold highlight in the app, but print as normal seats — no shading in the printed version.

🧹 Clean View hides all the ⋮/↻/🔒/undo icons on screen (great when arranging groups — all those buttons can get visually busy) without turning off dragging or seating — you can still place students while it's on. Toggle it off to get the icons back.
